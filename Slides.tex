\begin{frame}{Расследование утечек конфиденциальной информации}
\normalsize
\begin{block}{Data Leakage Prevention}
\normalsize
Data Leakage Prevention (DLP) – совокупность технологий предотвращения и/или расследования утечек данных.

\end{block}
\begin{block}{Проблема}
\normalsize
Необходимо расследовать утечки конфиденциальных текстовых документов, вызванные фотографированием экрана и последующей публикацией фотографий в публичных источниках.  
\end{block}
\begin{block}{Решение}
\normalsize
Встраивать идентификатор пользователя устройства в изображение документа, выводимого на экран. Встраивание идентификатора пользователя в изображение относится к технологии внедрения цифровых водяных знаков (ЦВЗ).
\end{block}

\end{frame}

\begin{frame}{Существующие решения}
\begin{block}{Методы маркирования текстовых документов}
\begin{itemize}
\small
\item Изменение длины пробелов между словами.
Биты кодируются соотношением суммарной длины половин пробелов в строке.\\
\tiny\textrm{\textit{Zou D., Shi Y. «Formatted Text Document Data Hiding Robust to Printing, Copying and Scanning», 2005}}
\item \small Слова в тексте заменяются синонимами в зависимости от кодируемого бита. Корректность замены проверяется с помощью Google n-gram corpus.\\
\tiny\textrm{\textit{Chang C.-Y., Clark S. «Practical Linguistic Steganography using Contextual Synonym Substitution and a Novel Vertex Coding Method», 2014}}

\item \small Добавление двух симметричных групп пикселей к буквам. Расстояние между группами кодируют бит. \\
\tiny\textrm{\textit{Varna A.L., Rane S. «Data hiding in hard-copy text documents robust to print, scan and photocopy operations», 2009}}

\end{itemize}
\end{block}
\begin{block}{Методы маркирования, устойчивые к атакам сценария screen-cam}
\begin{itemize}
\small
\item \small ЦВЗ внедряется в домен ДКП областей, задаваемых ключевыми точками, полученными с помощью алгоритма I-SIFT.\\ 
\tiny\textrm{\textit{Fang H., Zhang W.«Screen-Shooting Resilient Watermarking», 2019}}
\item \small Изменение яркости областей на экране. Биты кодируются уменьшением/увеличением яркости круговых областей.\\ 
\tiny\textrm{\textit{Gugelmann D., Sommer D.«Screen watermarking for data theft investigation and attribution», 2018}}
\end{itemize}
\end{block}
\end{frame}
%5--------------------------------------------------------------
\begin{frame}{Постановка задачи}
\large
Разработать и реализовать метод маркирования текстовых документов, отображаемых на экране монитора, а также метод извлечения ЦВЗ с фотографии экрана. Метод должен удовлетворять требованиям:
\begin{itemize}
\item Незаметность для пользователя устройства
\item Встраивание в режиме реального времени
\item Независимость от формата файла докумета
\item Устойчивость к атакам сценария screen-cam

\item Работа в режиме слепого извлечения (без оригинала документа)
\end{itemize}
\end{frame}
%6--------------------------------------------------------------
\begin{frame}{Разработанный подход к маркированию документов}
\begin{itemize}
\item ЦВЗ представляет собой шаблон, состоящий из прямоугольных областей разной яркости, называемых маркерами
\begin{itemize}
\item Светлые маркеры кодируют бит «0», темные --- «1»
\item Кодирующие маркеры чередуются с промежуточными для упрощения извлечения ЦВЗ
\item В начало, середину и конец шаблона встраиваются маркеры, состоящие из двух частей. Эти маркеры позволяют определить положение шаблона при извлечении ЦВЗ
\item На шаблон накладывается фильтр Гаусса для сглаживания переходов между маркерами, что снижает заметность ЦВЗ
\end{itemize}
\item Шаблон отображается при помощи частично прозрачного окна, лежащего поверх остальных окон
\item Шаблон встраивается в области межстрочных интервалов нижележащих окон
\end{itemize}

 \begin{minipage}[h]{0.48\linewidth}
\begin{block}{Алгоритм встраивания ЦВЗ}

\begin{enumerate}
\small
\item Создание снимка экрана
\item Очистка снимка экрана от предыдущей цифровой метки
\item Определение частей окон, видимых пользователем
\item Определение областей с текстом
\item Определение областей межстрочных интервалов
\item Генерация шаблона яркости, соответствующего  внедряемой информации
\item Встраивание шаблона на оверлей в области межстрочных интервалов маркируемого текста.

\end{enumerate}

\end{block}
\end{minipage}
\hfill
\begin{minipage}[h]{0.48\linewidth}
\begin{block} {Алгоритм извлечения ЦВЗ}
\vspace{0.05cm}
\begin{enumerate}
\small
\item Коррекция перспективы и обрезка фотографии
\item Определение областей с текстом
\item Определение областей межстрочных интервалов
\item Удаление из межстрочных интервалов фрагментов букв
\item Определение наличия ЦВЗ в межстрочных интервалах
\item Извлечение ЦВЗ из межстрочных интервалов
\item Объединение меток, извлеченных из разных межстрочных интервалов и текстовых областей.

\end{enumerate}

\end{block}

\begin{adjustwidth}{-0.9cm}{-0.9cm}

Маркеры цифровой метки могут быть встроены с разной непрозрачностью (opacity).

\item Непрозрачность 100\% --- пользователь видит на экране маркеры черного и белого цвета
\end{itemize}

\begin{frame}{Результаты работы алгоритма извлечения ЦВЗ}
\begin{block}

\footnotesize
Извлекаемое значение k-го бита сообщения зависит от яркости k-го кодирующего маркера.
\begin{itemize}
\item $GS_k$ --- оценка отклонения яркости маркера от среднего значения яркости в межстрочном интервале
\item $CS_k$ --- оценка выпуклости функции яркости около положения кодирующего маркера в межстрочном интервале 
\end{itemize}
Тогда общая оценка яркости кодирующего маркера:
$$S_k = GS_k + \alpha \cdot CS_k,\ \alpha > 0$$
\end{block}
\begin{block}

\footnotesize
Значение бита $m_k,\ k \in[1,N]$, соответствующего k-му кодирующему маркеру, вычисляется как: 
$$m_k = \begin{cases}0, & \text{если }S_k > 0;\\1, &\text{иначе.} \end{cases}$$
$|S_k|$ характеризует степень уверенности в извлеченном значении.
\end{block}
\begin{block}

\footnotesize
Bit Error Rate (BER) позволяет оценить успешность извлечения ЦВЗ. 
$$BER=\frac{\text{\{число неверно извлеченных бит цифровой метки\}}}{\text{\{общее число бит цифровой метки\}}}\cdot 100\%$$
\end{block}
